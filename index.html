Створіть файл index.html

Перейдіть на GitHub.com і створіть новий репозиторій або перейдіть до існуючого. Натисніть кнопку Створити новий файл.

Hello, World!

Назвіть файл index.html і введіть у редакторі деякий HTML-вміст.

Збережіть зміни

Прокрутіть донизу сторінки, напишіть повідомлення про копіювання та зафіксуйте новий файл (Commit new file).

Налаштування сховища

Натисніть вкладку Настройки та прокрутіть сторінку до розділу GitHub Pages. Потім виберіть джерело основної гілки та натисніть кнопку Зберегти.

Запустіть браузер і перейдіть на адресу http://username.github.io/, де username - ваш нікнейм.
